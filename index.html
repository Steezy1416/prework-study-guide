Git

    What is git?

  • Git is a open source version control system
  • Version control is a way to save and keep track of code changes made over time
  • A repository is centralized place where data or a project is stored
  • Git is the tool to keep track of chnages made and GitHub is a website to host your repositories

    • Git Commands

  • git clone: Gets you a local copy of an existing repository
  • git pull: Updates repository with new commits and the most updated version of the repository
  • git add: Adds the changes you want to make and are to the staging area
  • git commit: Saves the changes you want to make and is like a snapshot of your repo and over time keeps track of all the changes and different versions of your repo
  • git push: Uploads all the changes or commits you wanted to make to the remote repository
  • git status: Checks the current branch
  • git checkout -b "branch name": Creates and new branch and switches to it
  • Checkout command moves working branch to a new branch
  • -b flag makes a new branch
  • git branch -d "branch name": Deletes branch
  • git log: Show logs of all your commits
